### Software Development LifeCycle / SDLC

- It is a process used by software industry to design,develop and test high qaulity products.

- Every company while building their application they follow the following phases
- planning
- Defining
- Designing
- Building
- Testing
- Deploying

- **In planning**, we collect requirements from the customers

- **In defining**, we document our requirement in a document called SRD

- **In designing**, we used to discuss about HLD and LLD

- **In Building**, Developing the design models,start writing the application code,once he write the code he used to push the code into github,multiple developers collabrate with each others.

- **In Testing**, These application taken from github and test in a server by the QE team. 

- **In Deploying**, Promote the application into production server and customers receive our product.

- Devops improves the fasten the building,testing and deploying quickly by automation.

- Everyone is following Agile model in the Industry.
